# 시뮬레이션 시스템 가이드 (System Reference)

이 문서는 시뮬레이션 내의 자원 관리, 기술 그룹, 특성 로직을 정리한 통합 가이드입니다.

---

## 1. 기본 자원 및 생존 시스템

### **A. 능력치 한도 (Stat Caps)**
*   **체력 (HP):** 최대 **10**
*   **식량 (Food):** 최대 **30** (매일 소모됨)
*   **치료제 (Meds):** 최대 **10**
*   **은 (Money):** 최대 **30**

### **B. 일일 생존 로직**
*   **식량 소모:** 매일 시작 시 **식량 -1**이 고정적으로 소모됩니다.
*   **기아 상태:** 식량이 0인 상태에서 하루를 시작하면 **체력 -1** 페널티를 받습니다.

---

## 2. 특성 시스템 (Traits)

### **A. 이동 및 회피 관련 특성**
고정 확률(`fixedChance`)이 설정된 선택지에서 성공 확률에 보정을 줍니다.
*   **계산식:** `최종 확률 = Math.max(5, Math.min(95, 고정 확률 + 특성 보너스))`

| 특성 ID | 특성명 (KOR) | 성공 보너스 |
| :--- | :--- | :--- |
| `fast_walker` | **가벼운 발** | **+10%** |
| `jogger` | **신속** | **+10%** |
| `nimble` | **재빠름** | **+10%** |
| `slowpoke` | **느림보** | **-20%** |

### **B. 물리 방어 특성**
| 특성 ID | 특성명 (KOR) | 시스템 로직 |
| :--- | :--- | :--- |
| `tough` | **강인함** | **HP 피해량 50% 반감** (반올림 처리) |

## 3. 기술 그룹 (Skill Groups)

| 그룹명 | 포함 기술 | 용도 |
| :--- | :--- | :--- |
| **전투** | 사격, 격투 | 습격 대응, 사량 등 |
| **연구(지능)** | 지능 | 특수 수단, 분석 등 |
| **제작** | 건설, 제작, 채굴 | 시설 보수, 운석 채굴 등 |
| **재배** | 재배 | 농사, 수확 등 |
| **생존** | 재배, 조련 | 채집, 야생동물 대응 등 |
| **의학** | 의학 | 질병 치료, 부상 회복 등 |
| **사교** | 사교 | 거래, 협상, 방랑자 설득 등 |
