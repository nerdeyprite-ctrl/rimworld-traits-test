# 시뮬레이션 시스템 가이드 (System Reference)

이 문서는 시뮬레이션 내의 자원 관리, 기술 그룹, 특성 로직을 정리한 통합 가이드입니다.

---

## 1. 기본 자원 및 생존 시스템

### **A. 능력치 한도 (Stat Caps)**
*   **체력 (HP):** 최대 **10**
*   **식량 (Food):** 최대 **30** (매일 소모됨)
*   **치료제 (Meds):** 최대 **10**
*   **은 (Money):** 최대 **30**

### **B. 일일 생존 로직**
*   **식량 소모:** 매일 시작 시 **식량 -1**이 고정적으로 소모됩니다.
*   **기아 상태:** 식량이 0인 상태에서 하루를 시작하면 **체력 -1** 페널티를 받습니다.

---

## 2. 특성 시스템 (Traits)
특성은 시뮬레이션의 시작 조건이나 특정 상황에서의 확률, 피해량 등에 영향을 줍니다.

### **A. 이동 및 회피 관련 특성**
고정 확률(`fixedChance`)이 설정된 선택지에서 성공 확률에 보정을 줍니다.
*   **계산식:** `최종 확률 = Math.max(5, Math.min(95, 고정 확률 + 특성 보너스))`

| 특성 ID | 특성명 (KOR) | 성공 보너스 |
| :--- | :--- | :--- |
| `fast_walker` | **가벼운 발** | **+10%** |
| `jogger` | **신속** | **+10%** |
| `nimble` | **재빠름** | **+10%** |
| `slowpoke` | **느림보** | **-20%** |

### **B. 시작 물자 보너스**
특정 특성은 정착민의 초기 자원 보유량을 변화시킵니다. (기본: HP 5 / 식량 5 / 치료제 2 / 은 5)

| 특성 ID | 특성명 (KOR) | 시작 자원 변화 |
| :--- | :--- | :--- |
| `greedy` | **탐욕** | **은 +10** (총 15개로 시작) |
| `ascetic` | **검소** | **HP +5, 은 -5** (HP 10, 은 0으로 시작) |
| `wimp` | **엄살쟁이** | **치료제 +3** (총 5개로 시작) |

### **C. 전투 전용 로직**
| 특성 ID | 특성명 (KOR) | 시스템 로직 |
| :--- | :--- | :--- |
| `tough` | **강인함** | **HP 피해량 50% 반감** (반올림 처리) |

### **D. 전용 선택지 추가 특성**
아래 특성들은 특정 이벤트(습격, 질병, 평화로운 날 등)에서 **확률 보너스나 고유한 해결 방식**을 제공합니다.

| 특성 ID | 특성명 (KOR) | 전용 선택지가 발생하는 상황 |
| :--- | :--- | :--- |
| `tough` | **강인함** | 습격 시 정면 돌파 가능 (성공 확률 2배) |
| `wimp` | **심약자** | 습격 시 은신/도주 가능 (성공 확률 2배) |
| `industrious`| **근면성실** | 평화로운 날에 야근 (자원 추가 획득) |
| `lazy` | **게으름** | 평화로운 날에 완벽한 휴식 (상태 회복) |
| `kind` | **다정다감** | 교역이나 조우 시 호의 베풀기 |
| `abrasive` | **직설적** | 교역이나 조우 시 위협하기 |
| `pyromaniac` | **방화광** | 습격 시 화염병 투척 |

### **E. 현재 시뮬레이션 기능 미구현 특성**
아래 특성들은 현재 정착민 프로필에는 존재하나, 시뮬레이션 내 특수 로직은 아직 구현되지 않았습니다. (추후 업데이트 예정)
*   낙천주의자 / 비관주의자, 철의 의지 / 약한 의지, 사이코패스, 식인종, 피의 갈망, 나체주의자, 동성애자 등

## 3. 기술 그룹 (Skill Groups)
선택지 성공 확률은 관련 기술들의 **평균값**을 기준으로 결정됩니다.

| 그룹명 | 포함 기술 | 용도 |
| :--- | :--- | :--- |
| **격투/사격** | 사격, 격투 | 습격 대응, 사냥 등 |
| **의학** | 의학 | 질병 치료, 부상 회복, 휴식 효율 등 |
| **재배/생존** | 재배, 조련 | 농사, 수확, 조련, 은신 등 |
| **제작** | 건설, 제작, 채굴 | 시설 보수, 운석 채굴, 야근 효율 등 |
| **사교** | 사교 | 거래, 협상, 방랑자 설득 등 |
| **지능** | 지능 | 특수 수단, 분석, 연구 등 |

---

## 4. 기술 레벨별 기본 성공 확률
배율(`chanceMultiplier`)이 적용되는 특성 선택지는 아래 확률에서 **2배**가 적용됩니다. (최대 95% 제한)

| 평균 기술 레벨 | 성공 확률 |
| :--- | :--- |
| 0 ~ 3 | 30% |
| 4 ~ 6 | 60% |
| 7 ~ 10 | 80% |
| 11 이상 | 95% |
