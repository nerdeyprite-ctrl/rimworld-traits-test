# 림월드 정착민 테스트 데이터 가이드 (DATA_GUIDE.md)

이 문서는 `data/questions.json`을 수정하거나 새로운 질문을 추가할 때 참고하는 가이드라인입니다.

---

## 1. 질문 ID 규칙 (매우 중요)

이 테스트는 2단계로 진행되므로, **질문 ID(id)** 에 따라 등장 시점이 결정됩니다.

| ID 범위 | 단계 (Phase) | 목적 | 설명 |
| :--- | :--- | :--- | :--- |
| **1 ~ 99** | **1단계 (Phase 1)** | 기본 성향 파악 | 특성(Trait), MBTI, 백스토리 결정용 질문입니다. |
| **100 ~** | **2단계 (Phase 2)** | 스킬 정밀 분석 | "스킬 정밀 검사" 버튼을 눌렀을 때 나오는 심화 질문입니다. |

> **주의:** ID는 중복되면 안 됩니다. 새로운 질문을 추가할 때는 기존 마지막 번호 다음 번호를 사용하세요.

---

## 2. 점수 임계값 (Thresholds)

질문 답변을 통해 쌓인 점수가 일정 기준을 넘으면 **특성**이나 **스킬 열정**이 부여됩니다.

### A. 특성 (Trait) 획득 조건
*   **조건:** 특정 특성 키(Key)의 합산 점수가 **`3점` 이상**일 때 획득 후보가 됩니다.
*   **경합:** 같은 그룹(예: 아름다움 vs 흉측함) 내에서는 점수가 더 높은 쪽만 획득합니다.
*   **우선순위:** 점수가 높은 특성부터 우선적으로 부여됩니다.

### B. 스킬 열정 (Passion, 🔥) 조건
스킬 점수(답변 점수 + 백스토리 보너스) 합계에 따라 결정됩니다.

| 점수 합계 | 결과 | 표시 |
| :--- | :--- | :--- |
| **0 ~ 3점** | 열정 없음 | (없음) |
| **4 ~ 7점** | 약간의 열정 | 🔥 (Minor) |
| **8점 이상** | 불타는 열정 | 🔥🔥 (Major) |

### C. 스킬 레벨 (Level)
*   기본 공식: `0~3(랜덤) + 답변 점수 + (백스토리 보너스 / 2)`
*   최대 레벨은 **20**, 최소 레벨은 **0**으로 고정됩니다.

## 3. MBTI 산출 로직

MBTI는 별도의 질문이 있는 것이 아니라, **쌓인 스킬 점수**를 기반으로 계산됩니다.

| 지표 | 결정 방식 (높은 쪽의 성향 획득) | 관련 스킬 |
| :--- | :--- | :--- |
| **E / I** | **E(외향):** 사교(1.5배) + 조련 <br> **I(내향):** 연구 + 예술 | Social, Animals, Intellectual, Artistic |
| **N / S** | **N(직관):** (연구 + 예술) × 1.2 <br> **S(감각):** 원예 + 채굴 + 건설 | Intellectual, Artistic, Plants, Mining, Construction |
| **T / F** | **F(감정):** 의학 + 사교 + 조련 <br> **T(사고):** 사격 + 격투 + 제작 | Medicine, Social, Animals, Shooting, Melee, Crafting |
| **J / P** | **J(판단):** 건설 + 조리 (+근면함 특성) <br> **P(인식):** 사격 + 격투 (+게으름 특성) | Construction, Cooking, Shooting, Melee |

## 4. 데이터 작성 템플릿 (JSON)

`data/questions.json` 파일에 아래 형식을 따라 추가하세요.

```json
{
    "id": 105,
    "text": "새로운 질문 내용을 적으세요.",
    "answers": [
        {
            "text": "선택지 1번 내용",
            "scores": {
                "shooting": 2,      // 스킬: 사격 +2점
                "brave": 1          // 특성: 용감함 +1점
            }
        },
        {
            "text": "선택지 2번 내용",
            "scores": {
                "artistic": 3,      // 스킬: 예술 +3점
                "ascetic": 2        // 특성: 금욕적 +2점
            }
        }
    ]
}
```

## 5. 사용 가능한 키 (Keys) 참고

### 스킬 키 (Skills)
*   **전투:** `Shooting` (사격), `Melee` (격투)
*   **건설/채굴:** `Construction` (건설), `Mining` (채굴)
*   **생활:** `Cooking` (조리), `Plants` (원예), `Animals` (조련)
*   **전문:** `Crafting` (제작), `Artistic` (예술), `Medicine` (의학)
*   **지식/사회:** `Social` (사교), `Intellectual` (연구)

### 특성 키 (Traits) - 예시
*   `data/traits.json` 파일의 `id` 값을 사용합니다.
*   주요 예시: `psychopath` (사이코패스), `cannibal` (식인), `kind` (다정함), `brave` (강인함), `wimp` (유리멘탈), `greedy` (탐욕), `jealous` (질투) 등.
